# Projet de fin d'études - Gilet connecté

## Introduction
Le projet que nous entreprenons s'inscrit dans le but d'inciter les manutentionnaires à gérer convenablement leurs mouvemnets et diminuer les situations à risques. Nous travaillerons sur un gilet connecté permettant d'envoyer des données à analyser, pour l'étude des gestes d'un utilisateur.
Ce produit pourra être proposé aux assurances qui intégreront cette technologie dans leur formule.
